# 酒桌小游戏集合 H5 开发计划 (含左轮模拟器详解)

本计划旨在创建一个纯前端、可免费托管的酒桌游戏 H5 应用。我们将重点打造“左轮手枪模拟器”，确保其在不依赖外部复杂素材的情况下，依然拥有出色的交互体验。

## 核心策略：纯静态开发 + 免费托管 + 代码绘图

* **部署**: 采用纯静态架构，支持 GitHub Pages 一键免费部署。

* **素材**: 核心图形将使用 **CSS3 和 SVG 代码绘制**，**您无需提供图片**即可直接运行。同时也支持您后续替换为真实图片。

## 详细功能方案：左轮手枪模拟器

我们将摒弃复杂的 3D 建模，采用\*\*“第一人称弹巢视角”\*\*，专注于紧张感的营造。

### 1. 界面设计 (Visuals)

* **核心视图**: 屏幕中央展示一个巨大的**左轮手枪弹巢 (Cylinder)** 横截面。

  * **弹孔**: 6 个圆孔呈环形排列。

  * **子弹**: 可视化显示。点击弹孔可切换“有弹/无弹”状态（黄铜色圆形代表子弹）。

  * **击针指示**: 顶部固定一个倒三角指针，指示当前即将击发的弹孔位置。

* **风格**: 深色金属质感 UI，营造紧张氛围。

### 2. 交互流程 (Interaction)

* **阶段一：装弹 (Loading)**

  * **手动模式**: 点击任意空弹孔，填入一颗子弹。

  * **随机模式**: 点击“随机一颗”按钮，系统自动在随机位置塞入一颗子弹。

* **阶段二：旋转 (Spinning)**

  * **操作**: 点击“旋转弹巢”按钮或滑动屏幕。

  * **反馈**: 弹巢高速旋转，伴随“咔哒咔哒”的转动声，最终随机停在某个位置。此时**隐藏子弹视图**（变为不可见模式，增加盲射刺激感）。

* **阶段三：射击 (Firing)**

  * **操作**: 点击底部巨大的红黑配色“开火”按钮。

  * **判定**:

    * **情况 A (空枪)**: 播放清脆的金属撞击声 ("Click")。弹巢自动逆时针转动一格，准备下一发。

    * **情况 B (中弹)**:

      * **视觉**: 屏幕剧烈闪烁红光/白光，模拟开枪后坐力震动。

      * **听觉**: 播放响亮的枪声 ("BANG!")。

      * **结果**: 弹出“你挂了/饮酒一杯”的提示框。

### 3. 技术实现细节

* **状态管理**: 使用一个数组 `[0, 1, 0, 0, 0, 0]` 记录子弹位置，使用指针 `currentIndex` 记录当前位置。

* **动画**: 使用 CSS `transform: rotate()` 配合 `transition` 实现流畅的旋转动画。

* **音效**: 使用 HTML5 `<audio>` 标签。我们将提供基础的开源音效文件，或者使用 Web Audio API 合成简单的提示音，确保无素材也能运行。

## 开发路线图

1. 初始化 Vue 3 + Vite 项目。
2. 配置 `vite.config.js` 以支持相对路径部署 (GitHub Pages 适配)。
3. 建立 `Home` (大厅) 和 `Revolver` (左轮) 页面路由。

2. **实现逻辑**: 编写装弹、旋转、开火的 JavaScript 逻辑。
3. **添加音效**: 集成基础音效逻辑。
